    1. pip install pyspice
    2. pip install opencv-python for image output
    3. install ngspice dll libraries
      1. installation: pyspice-post-installation --install-ngspice-dll
      2. check: pyspice-post-installation --check-install

Advanced Power Resources

Behavioral Resource

spice syntax:

BXXXXXXX n+ n- <i=expr> <v=expr> <tc1=value> <tc2=value> <temp=value> <dtemp=value>

functions: The following functions of a single real variable are defined:

  • Trigonometric functions: cos, sin, tan, acos, asin, atan
  • Hyperbolic functions: cosh, sinh, acosh, asinh, atanh
  • Exponential and logarithmic: exp, ln, log, log10 (ln, log with base e, log10 with base 10)
  • Other: abs, sqrt, u, u2, uramp, floor, ceil, i Functions of two variables are min, max, pow, **, pwr, ^
  • Functions of three variables are a ? b:c

Refer to page 102 of ngspice-35-manual

Keyword Parameters:

  • current_expression / i
  • voltage_expression / v
  • tc1: first order temperature coeff.
  • tc2: second order temperature coeff.
  • temperature / temp
  • device_temperature / dtemp

special variable available:

  • time, 0 in AC analysis
  • temper, given in degree C
  • hertz, 0 in transient analysis

example:

circuit.B('1', 'p1', circuit.gnd, voltage_expression='exp(time)')
